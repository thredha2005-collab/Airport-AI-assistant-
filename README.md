# ✈️ AI Airport Assistant

An AI-powered airport operations and passenger assistance platform that combines Artificial Intelligence, Machine Learning, Analytics, and Full-Stack Web Development to improve airport operations and enhance passenger experience.
## 📖 Project Overview

The AI Airport Assistant is an intelligent airport management platform developed to improve operational efficiency and provide passengers with real-time information.

The system integrates AI, predictive analytics, and interactive dashboards to simulate airport operations including flight information, passenger services, parking availability, weather updates, facility management, and AI chatbot assistance.
## ✨ Features

- AI Chatbot for passenger assistance
- Flight Information Dashboard
- Parking Availability Monitoring
- Passenger Portal
- Weather Information
- Facility Management
- Live Operational Dashboard
- Predictive Analytics using Machine Learning
- Simulation Engine
- Secure Login System

## ⚙ Installation

Clone the repository

git clone https://github.com/thredha2005-collab/Airport-AI-assistant-.git

Frontend

cd frontend

npm install

npm run dev

Backend

cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
